水走顶的优劣：
- 便于及时发现漏水，维修成本低
- 不会影响楼下
- 有利于卫生间防水
走顶热水管需包裹保温棉，保温兼防冷凝水滴落
水走地的优劣：
- 不用打孔穿梁，成本更低
- 墙面开横槽？
厨卫水电直接走顶。

排污管管径110mm，卫生间沉箱支管、厨房管径75mm，阳台排水管管径50mm。
排水管三通考虑斜三通。
![三通](https://github.com/zhangyunwu/decoration/blob/main/pic/%E6%B0%B4%E7%AE%A1%E4%B8%89%E9%80%9A.jpg)

洗手盆、洗菜盆、蹲便、地漏等都需要布置存水弯。

通水试压要求0.8MPa半小时压降不超过0.05MPa，且所有管道阀门、接头无渗漏。冷热水管最好串联加压。

卫生间要注意门槛石位置的防水，一般水管不从门槛石下方走。

完工后下水管的口子需要用盖子封好，避免掉进杂物。

前置过滤器。

## 瓷砖
术语：全瓷、抛釉、加工砖

抛釉砖更便宜。
全瓷砖比抛釉砖更耐用，地砖最好用全瓷砖，卫生间也用全瓷砖，厨房可用抛釉砖。


全瓷砖可先少买，不够再补货；抛釉砖可稍微多买，有剩可退。加工砖有剩没法退。
### 墙砖
阳角做海棠角。阳角倒角，由瓷砖厂家直接用机器做，瓦工现场倒角容易崩瓷。

瓷砖开孔需用开孔器。

铺贴完成后需对管、线位置做好位置标识。
### 地砖

卫生间用防滑小砖，贴砖最好墙地对缝铺贴，壁龛也需要对缝。

地砖缝隙2mm，800mm\*800mm地砖工价重庆咸鱼40+，应能谈到30+。800mm\*800mm瓷砖单价30-40一片

地砖铺贴完毕后需做瓷砖保护。

## 定制家具
颗粒板、密度板、多层板
颗粒板属于比较差的板材，密度板比颗粒板好但是密度板很重，对五金要求高。可考虑多层板，但多层板似乎容易变形。 
- PET门板
PET板就是基板加膜，主要关注点还是在基板。

基板方面，两边用颗粒板，中间夹多层板。颗粒板能实现很高的平整度。PET门板对封边要求高，封边不好显得low

实木多层市场价200-230/平，不包安装不包测量；密度板市场价160-170
- 吸塑门板
吸塑门板相比PET门板工艺更复杂

衣柜板材用杉木芯的实木免漆生态板比较好。一线品牌单价230，二线品牌单价180。定制用一线品牌单价可能1000左右，直接找板材店定制可能800左右。

PVC吸塑板受晒后容易起皮，不能用在阳台。

## 冰箱
如果不是底部散热，左右就需要留出5-10cm空间用于散热。除了散热还需要考虑冰箱门的开启，是否需要额外空间。

法式四门冰箱的底部抽屉体验较好，但低价位一般为对开门冰箱。

尽量买一级能效标识

风冷冰箱不结霜，但容易让食物变干

单循环、双循环：双循环冰箱的冷藏室和冷冻室各有一个蒸发器，冷藏室和冷冻室独立控温，且可以避免串味

压缩机：冰箱压缩机一般混用，不同批次可能是不同品牌的压缩机。一般体验差别不大，选择变频压缩机就好。

保鲜技术：蔬菜在不同温湿度环境保鲜时长不同，
