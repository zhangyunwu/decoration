### 保温防水
防水最好找专业防水的，不找瓦工做。防水牌子：雨虹、德高、广州双虹、科顺。

卫生间水管要用红砖围起来。

墙、地接缝处，管道、地面接缝处等边角位置是最容易出问题的地方，防水涂料一定要做到位。

卫生间、阳台防水施工步骤
```
1、基层清理

先用铲刀和笤帚及工具将基层表面的突起物、砂浆和疙瘩等铲除并将尘土杂物清扫干净，对阴阳角、管根、地漏等部位应认真清理。

2、涂刷底面防水层

由专人负责称取材料配制，按上表提供的配合比分别称出配料所用的液料、粉料、用手提电动搅拌器搅拌均匀，使其不含有未分解的粉料。用滚刷或油漆刷均匀地涂刷底层，不得漏底。待涂层干固后，才能进行下道工序。

3、细部附加层

对地漏、管根、阴阳角等易发生漏水的部位，应进行特殊加强处理。细部附加层：在地漏、管根、阴阳角和出入口等易发生漏水的薄弱部位，可增加一层刚性防水，刚性防水宽度应不小于100mm，搭接宽度应不小于100mm。然后涂刷福窝专用防水涂料。

4、涂刷中间防水层

将配制好的防水涂料均匀涂刷在基层表面，涂刷量以 0.8—1.0kg/㎡每遍为宜，直至防水合格为止，家装防水要求一般为1.0mm左右。

(1) 闭水试验

在最后一遍防水层干固24h后，蓄水48h，无渗漏为合格。

(2) 饰面层施工

蓄水试验合格后，即可做饰面层施工。

卫生间做防水也有讲究，按照相关规范的要求
```
Ø 淋浴墙及相邻的两个墙体需做不低于1.8米的高度；
Ø 洗手盆部位不低于1.2米；
Ø 其它区域不低于0.5米；
Ø 立面墙与地面墙的交接部位——阴角、管道与地面的交接部位需要做加强处理；门口部位需要做挡水台阶。因为我们淋浴时水汽会上升到防水涂刷高度以上，造成迎水墙体有渗水现象，出现发霉有水污的现象。为了达到更好的防水效果，雄大在这里建议卫生间最好做全墙防水方案，所有墙面涂刷至顶。在选材方面，如果是新建房屋，考虑到楼体有沉降的因素，建议大家选择柔性防水涂料。如果墙地面起灰、起砂、坚实程度不够，需要用加固剂做基层套胶处理。需要注意的是防水层的厚度，按照相关规范要求：
Ø 墙面防水涂膜厚度不低于1.2mm，地面防水涂膜厚度不低于1.5mm。
Ø 在瓷砖铺贴时，应根据购买的瓷砖类型，去选择相应的粘结材料，建议在装修时咨询专业人士。开发商做了防水有必要再做防水吗？一般来说，开发商做过防水是不需要重新做的，但也有难以预料的事，比如选用的低劣材质、施工不规范，这会导致入住一两年后出现渗水现象。为了安全起见，建议铲除原防水至基层，在基层上重新涂刷防水涂料。

## 厨房
烟道要用红砖包，不然高温可能导致瓷砖开裂掉落。烟道止逆阀自己买质量好一点的，安装止逆阀不能直接打胶，需要钻孔用螺丝固定、同时用胶密封。

- 油烟机与吊柜
油烟机挂式安装，如不注意会让油烟机和吊柜底部出现3cm缝隙。先装油烟机再装吊柜可以解决，但如果油烟机需要拆卸检修就得先拆吊柜。可考虑先装吊柜柜体，后装油烟机，最后装柜门的方式来消除缝隙，并且油烟机拆卸时只需要拆柜门。

## 瓷砖
术语：全瓷、抛釉、加工砖
- 瓷片：吸水率>10%，标准名为陶质砖，原料为低硅低铝高铁的粘土，烧制温度在1000℃以下。

- 玻化砖：吸水率<0.5%，标准名为瓷质砖，原料为高硅高铝低铁或高硅高铝高铁的瓷土，1200℃以上温度烧制，硬度更高。

- 抛光砖：通体砖经过机械研磨、表面抛光的陶瓷砖。表面光亮，不耐脏。抛光砖表面有细微小孔，有水时反而更防滑，可考虑用于浴室地面。

- 釉面砖：

- 抛釉砖：胚体+底釉+色釉+透明釉。抛釉砖硬度较高，防滑度相对较低。抛釉能做出各种花纹。可考虑用于客厅卧室。


- 仿古砖：仿古砖属于釉面砖。

- 全抛釉砖：

显然，前几种属于无釉砖，后几种属于釉面砖。釉面砖抗污易清洁。
玻化砖与抛光砖不等同，玻化指的是砖体，抛光指的是砖面。
抛光砖、仿古砖、玻化砖应该都算全瓷砖。
价格上，抛釉砖>釉面砖>抛光砖。

瓷片吸水率高、易上墙，铺贴前需泡水。

抛釉砖更便宜。
全瓷砖比抛釉砖更耐用，地砖最好用全瓷砖，卫生间也用全瓷砖，厨房可用抛釉砖。

## 冰箱
如果不是底部散热，左右就需要留出5-10cm空间用于散热。除了散热还需要考虑冰箱门的开启，是否需要额外空间。

法式四门冰箱的底部抽屉体验较好，但低价位一般为对开门冰箱。

尽量买一级能效标识

风冷冰箱不结霜，但容易让食物变干

单循环、双循环：双循环冰箱的冷藏室和冷冻室各有一个蒸发器，冷藏室和冷冻室独立控温，且可以避免串味

压缩机：冰箱压缩机一般混用，不同批次可能是不同品牌的压缩机。一般体验差别不大，选择变频压缩机就好。

保鲜技术：蔬菜在不同温湿度环境保鲜时长不同，
![冰箱保鲜技术总结](https://github.com/zhangyunwu/decoration/blob/main/pic/%E5%86%B0%E7%AE%B1%E4%BF%9D%E9%B2%9C%E6%8A%80%E6%9C%AF%E6%80%BB%E7%BB%93.png)

抑菌除味，最好有这方面的配置
